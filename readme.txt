== Description ==
This plugin is an addon of Elementor Page Builder. A simple and nice Contact Form 7 Widget for elementor.No need of going in cf7 & copying the shortcode. It's help you to add contact form 7 drag and drop. You can change 
form style using this widget.

For more details :[CF7 for Elementor](https://wpocean.com/wp/plugins/cf7).
For our other elementor & WP works visit : [wpOcean](https://wpocean.com). 


== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
